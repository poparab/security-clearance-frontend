أزرار الأيقونات — Icon Buttons (System Standard)

Standard Icon Buttons — All Blue

جميع أزرار الأيقونات في النظام تستخدم نفس اللون الأزرق — جميعها بدون استثناء. الحذف والتعديل والحفظ والعرض كلها باللون الأزرق.
All icon buttons use the same blue-on-blue color. Delete, edit, save, view — all blue. No exceptions.

قاعدة مهمة — Critical Rule

جميع أزرار الأيقونات (حذف، تعديل، حفظ، عرض) تستخدم نفس اللون: خلفية #eaf4ff وأيقونة #005FCC. لا توجد أزرار حمراء للحذف في هذا النظام.
ALL icon buttons (delete, edit, save, view) use the same color: bg #eaf4ff + icon #005FCC. There are NO red delete buttons in this system.
