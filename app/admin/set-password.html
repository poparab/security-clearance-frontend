تعيين كلمة مرور دائمة

هذه الخطوة إلزامية للحسابات الجديدة من نوع Admin أو Operator قبل فتح أي صفحة داخل لوحة الإدارة.

  • لا يمكن استخدام كلمة المرور المؤقتة نفسها.
  • يتم حفظ كلمة المرور الجديدة قبل السماح بالوصول.
  • بعد النجاح تستمر الجلسة ويصل المستخدم إلى الشاشة المناسبة لدوره.
